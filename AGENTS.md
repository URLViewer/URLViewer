# 回答言語およびプロジェクトドキュメント保存ルール

## 基本方針

- 特に指示がない限り、日本語で回答する。  コミットメッセージ生成も。コミットは自分でやるのでやらないで。
- 以下のドキュメントも日本語で作成する：
    - 実装計画 (Implementation Plan)
    - 修正内容の確認 (Walkthrough)
    - タスクリスト (Task List)
- nodeのパッケージインストーラーには**yarn**を使用する。

## ファイル構成

- チャット内で作成した以下の3つのファイルを保存する：
    - `task.md`
    - `implementation_plan.md`
    - `walkthrough.md`

## 保存場所

- プロジェクト内の `docs` フォルダの下に新しいフォルダを作成する。
